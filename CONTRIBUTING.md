# Contributing

Found a mistake, unclear explanation, or factual error? We'd love your help.

## How to contribute

1. **Open an issue** describing the problem (course, unit, lesson, and what's wrong)
2. **Or submit a PR** with the fix — edit the relevant unit markdown file under `courses/`

## Guidelines

- Keep explanations concise — one clear sentence beats three vague ones
- Preserve the LaTeX math formatting (`$inline$` and `$$block$$`)
- Don't add practice questions — the sample previews are auto-generated from the full question bank on [tryals.app](https://tryals.app)
- Attribution: all contributions are licensed under [CC BY-SA 4.0](LICENSE)

## Reporting issues

If something doesn't render correctly on GitHub (broken LaTeX, formatting issues), please open an issue with a screenshot.
